/api/{ver}/servers - GET (Получение списка серверов и кол. серверов)
                   - POST (добавление сервера)

/api/{ver}/servers/{id} - GET (подробная информация о сервере)
                        - PUT (обновление инфы о сервере)
                        - DELETE (удаление сервера из списка)


/api/{ver}/rooms - GET (Получения списка публичных комнат и их количество)
                 - POST (Добавление комнаты)

/api/{ver}/rooms/{id} - GET (Подробная информация о комнате)
                      - PUT (обновление инфы о комнате)
                      - DELETE (Удаление комнаты)
